Award your users for their birthday.

== Description ==

Birthdays lets you award your users for their birthday.

For example, you can award 100 points on each user birthday and award an achievement to celebrate the user 18th birthday.

In addition, this add-on comes with built-in integration for BuddyPress and BuddyBoss profile fields.

Note: This add-on is designed to work based on the user birthday. To award for the account registration date, please check the GamiPress - Anniversaries add-on.

= New Events =

* Your a birthday: When is the user birthday.
* Your specific birthday: When is a specific user birthday (18th birthday, 25th birthday, etc).

= Features =

* Reward your users to celebrate their birthday.
* Ability to configure rewards based on the user age (18th birthday, 25th birthday, etc).
* Users will get awarded on their birthday without login thanks to a background processing to determine the user birthday and its age.
* Configure the user meta where the user birthday gets stored.
* Includes integration for BuddyPress and BuddyBoss profile fields.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
