Connect GamiPress with The Events Calendar

== Description ==

Gamify your [The Events Calendar](http://wordpress.org/plugins/the-events-calendar/ "The Events Calendar") events thanks to the powerful gamification plugin, [GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress")!

In addition, this integration also connects with the [Event Tickets](http://wordpress.org/plugins/event-tickets/ "Event Tickets") plugin to let you gamify events attendance and ticket purchases.

This plugin automatically connects GamiPress with The Events Calendar adding new activity events.

= New Events =

* New event: When an user publish a new event.
* Delete event: When an user deletes an event.

= [Event Tickets](http://wordpress.org/plugins/event-tickets/ "Event Tickets") Events =

* Confirm RSVP for an event: When an user confirms RSVP for an event.
* Confirm RSVP for a specific event: When an user confirms RSVP for a specific event.
* Purchase a ticket for an event: When an user purchases a ticket for an event.
* Purchase a ticket for a specific event: When an user purchases a ticket for a specific event.
* Check-in at an event: When an user check-ins at an event.
* Check-in at specific event: When an user check-ins at specific event.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
